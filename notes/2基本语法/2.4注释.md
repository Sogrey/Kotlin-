# 2.4注释

注释方便了我们阅读代码，与 java 和 javaScript 一样，Kotlin 支持单行注释和块注释。

	// 单行注释
	/* 哈哈哈哈
	   这是块注释 */
    /**
     * 用于记录Kotlin代码的语言（相当于Java的JavaDoc）称为KDoc
     */

与 java 不同的是 Kotlin 的 块注释可以级联。

关于Kotlin文档化注释看[这里](http://kotlinlang.org/docs/reference/kotlin-doc.html).
